Hello There!


My name's Evan, and I'm a graduate of General Assembly's Web Development Intensive. While at General Assembly I spent every single day for 3 months coding. Check Out Some Projects!

Casting Frontier

Casting Frontier is a digital casting company with web applications written in Ruby and PHP, and Apple/iOS/Windows/Android applications. The web application and downloadable applications all utilize a large scale Mysql database, Redis database, Mongo database, and added deeper search functionality via Elasticsearch. As a current web developer for Casting Frontier I have built new features from the backend to the frontend, fixed bugs all across the platform and handled deployment and server issues. I have written numerous tests for the front and back end in the Ruby test suite. I learned about custom server setup and deployment, learned how to utilize and endless string of new tools, gems and libraries, and have learned how to improvise to solve problems on all aspects of a very unique and customized application.

Tripwing

Tripwing is a business to business travel application. It's purpose is to provide a simple to use, heavily autimated itinerary builder for travel agents and tour providers. Starting on day one I was building out new features, updating and maintaining older features, implenting new technologies, writiing full feature tests and many other tasks. My responsibilites ranged from Full Stack Developer, UI/UX Designer, to Customer Service Representative, and Company Rep at Client Meetings. I have been involved in all aspects of this application and business from day one.

Split.It

Split.It is an app designed to handle the issue of splitting costs on group trips using an algorithm written by two of my teammates. On this project I handled the dynamic jQuery calendar which renders the total time of the trip and each users time on the trip. I also was in charge of the github repo and handled at merge conflicts, trouble shooted all aspects of the project, and wrote Capybara tests.

Band Buddy

Band Buddy is an app designed to simplify the process of bands looking for musicians, and musicians looking for bands. Using a questionnaire on music preferences and an algorithm which I wrote each users answers are compared to the current users answers and a percentage of matchability is rendered. The project was completed over a 5 day project sprint. Features which will be implemented include messaging between users, SoundCloud and Band Camp API's, and an entire reworking of the front end design.

Hipper Chat

Hipper Chat is a simple chat room app built using Node.JS, ExpressJS, and Socket.io. It was built as part of a group 7 hour hack-a-thon designed to gain experience using new technologies. In the allotted time we were able to research our new technology, implement two frameworks and create a simple chat room which displays each users name, and when users enter and leave the chat.

Fantasy Football Dashboard

My fantasy football dashboard was the first project built at General Assembly. It is a very basic dashboard app that calls on multiple APIs and RSS feeds depending on user selection, and renders specific content from each. With this project I built my own (very basic) user authentication process, made numerous API and RSS feed calls, iterated through large stores of data and rendered topical information based on user input.
